Set up Jobhunter

Follow the steps below to get the scraping pipeline running locally with Docker.

01

Add your Groq API key

Open the .env file in the project root and paste your Groq API key. Groq provides access to extremely fast AI models, and its free tier is sufficient for this project. Don’t have an API key yet? Get one at console.groq.com.

You can use it without that but you are going to lose AI job analysis.

If you don't need it, skip this step.

02

Start the containers

Run docker compose up --build in your terminal to build and start all services.

03

Open the interfaces

Go to localhost:5001 for the Flask application and localhost:8080 for Airflow. Default Airflow credentials: admin / admin.

04

Start scraping

Click the Play button in the Flask application to start the scraping pipeline. Results will appear in the Jobs tab as they are collected.

By default, the pipeline runs every hour. You can change the schedule at any time in the Airflow UI at localhost:8080.
